# Jamf PAW for PowerShell - A Jamf Pro API Wrapper for Simplified Script Automations (PowerShell)

This is a statically defined, standalone wrapper for JAMF Pro API functionality that serves as a vendor for larger MDM automations.

The focus of this implementation is on simplifying, demistifying and standardizing JAMF API calls across a highly cohesive PowerShell interface accessible to most developer skill-levels.

This implementation uses a combination of REST API endpoints from 'Jamf Classic API' and 'Jamf Pro API' references.

Originally developed in Western Australia. 

This repository contains modified code which I originally developed and implemented while working in the education sector, tested with a fleet of over 1.5k+ Apple Silicone devices.

# Pre-Requisites

- An active Jamf Pro license.
- An Active Jamf Pro instance accessible through a `jamfcloud` URL.
- Jamf API administrator credentials.

# PowerShell Requirements

Host machine must have an installation of **PowerShell v2025.2.0** or greater.
